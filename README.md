# DreamsFinal

 Pet-project. Application for recording dreams, sorting by mood, tags, lucid/not lucid.
 All user content is stored using Firebase
 
 Using technologies:
* Kotlin
* Jetpack Compose
* Dagger Hilt
* Coroutines
* MVVM
* Firebase (Google Auth and Firestore)
